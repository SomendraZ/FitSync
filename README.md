# 💪 FitSync

**FitSync** is a full-stack fitness tracking application built using **React Native**, **Flask**, and **MongoDB**.
It helps users track workouts, monitor nutrition, and analyze fitness progress through a mobile-first experience.

---

## 🚀 Features

* 🔐 User Authentication (JWT-based)
* 🏋️ Workout Tracking (sets, reps, exercises)
* 🍽️ Nutrition Logging (calories & protein)
* ⚖️ Progress Tracking (weight & history)
* 📊 Analytics (planned)

---

## 🏗️ Tech Stack

### 📱 Frontend

* React Native

### ⚙️ Backend

* Flask (Python)

### 🗄️ Database

* MongoDB

### 🔐 Authentication

* JWT (JSON Web Tokens)
